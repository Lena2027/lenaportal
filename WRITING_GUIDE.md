# Germany Blog 글 작성 가이드

이 가이드는 `lena-var4.com/germany/` 블로그에 새로운 글을 추가할 때 참고하는 매뉴얼입니다.
복잡한 HTML, CSS 코드를 만질 필요 없이 마크다운(`*.md`) 파일 하나만 작성하면 모든 디자인과 SEO 설정이 자동으로 렌더링됩니다!

## 1. 글 파일 생성 위치

새로운 글은 반드시 다음 경로에 생성해야 합니다:
```text
_posts/germany/YYYY-MM-DD-원하는-영어-제목.md
```
예시: `_posts/germany/2026-03-20-berlin-trip.md`

## 2. 필수 머리말 (Front Matter) 작성

문서 가장 상단에 아래와 같이 `---` 로 둘러싸인 메타데이터 구역을 반드시 먼저 작성하세요.

```yaml
---
title: "글의 실제 제목을 적어주세요"
description: "목록 카드와 구글 검색결과에 노출될 설명입니다."
date: 2026-03-20
theme: default
thumbnail: /assets/images/germany/폴더명/썸네일.jpg
tags: [날씨, 여행, 독일생활 중 택 1~N]
lang: ko
layout: germany/post
---
```

### 각 항목 설명

- **title** : 이 블로그 글의 실제 제목입니다.
- **description** : 1~2줄로 요약한 본문 설명입니다. 목록 카드와 구글 검색 결과, SNS 공유 시 나타납니다.
- **date** : 글의 작성 날짜입니다 (`YYYY-MM-DD` 형식 유지).
- **theme** : 글마다 다른 배경/분위기를 지정할 수 있는 마법의 키워드입니다! (밑에서 자세히 설명합니다.)
- **thumbnail** : 목록 카드와 링크 공유 썸네일에 들어갈 대표 이미지 경로입니다. 설정하지 않으면 자동으로 기본 썸네일 (🇩🇪 국기 이모지) 이 할당됩니다.
- **tags** : 연관 태그들을 써주세요. `_data/germany_tags.yml`에 있는 태그 ID나 아무 텍스트나 가능합니다.
- **lang** & **layout** : 독일 블로그 레이아웃과 한국어 설정을 위해 그대로 둡니다.

## 3. 마법의 테마 시스템 (theme)

현재 준비된 테마는 총 5가지입니다! 글의 내용에 맞춰 `theme: ` 뒤에 원하는 키워드만 적어주세요.

* `default` — 기본적으로 사용되는 깔끔한 흰 배경 테마입니다.
* `weather` — 하늘색 그라디언트 배경 위로 귀여운 구름 애니메이션이 둥둥 떠다닙니다. 비 오는 날이나 날씨 얘기에 좋습니다.
* `cozy` — 베이지, 브라운 톤의 따뜻하고 포근한 느낌입니다. 일상이나 카페 얘기에 찰떡!
* `city` — 네온 포인트와 다크 모드가 결합된 어두운 느낌의 세련된 테마입니다.
* `food` — 오렌지/레드 계열의 따뜻한 느낌으로, 맛집이나 레시피 소개에 딱 맞습니다.

## 4. 본문 작성

`---` 가 끝난 부분 아래쪽부터 마크다운 문법으로 자유롭게 글을 작성하시면 됩니다!
이미지 등은 `![](/assets/images/...)` 식으로 넣으시면 모바일과 데스크톱 모두 화면에 알맞게 크기가 자동 조정됩니다.

## 5. 완료 후

파일을 저장하고 Jekyll 빌드를 실행해주면 자동으로:
1. 홈페이지 진입 시 오른쪽 사이드바에 '최신 글'로 등록됩니다.
2. 메인 글 목록 피드에 카드가 예쁘게 등록됩니다.
3. 구글 및 SNS 검색에 유리한 SEO 메타 데이터들이 자동으로 생성됩니다!
